Controls:
Game
Esc: quit
Hero
WASD: move
F: print location to console
Lighting
Number keys 0-3: Select corresponding light source
Arrow keys: move selected light
Z/X: Increase/Decrease light Z coord
C/V & B/N: Increase/Decrease selected light near/far cutoff
K/L: Increase/Decreas selected light intensity
H: Toggle selected light on/off
Material
Number keys 5-6: select left minion & hero
Number keys 7-9: select the ambient, diffuse, and specular property of the selected object
E/R T/Y U/I: Increase/decrease R, G, and B channels of selected property
O/P: Increade/decrease the shininess of selected property
